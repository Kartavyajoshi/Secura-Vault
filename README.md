**SecuraVault** is a lightweight, secure, and completely offline password management application. It is designed for users who want to keep their digital life organized without trusting their data to the cloud.

## ğŸ“¥ How to Use SecuraVault

1.  **Download:** Go to the [**Releases**](https://github.com/Kartavyajoshi/Secura-Vault/releases) section and download the latest `SecuraVault.exe`.
2.  **Setup Folder:** Place the `.exe` file in a dedicated folder (e.g., `Documents/SecuraVault`).
3.  **Launch:** Double-click the file to start managing your passwords.
4.  **Important:** A file named `vault.db` will be created in the same folder. **Do not delete this file**, as it contains all your saved passwords.

> [!TIP]
> **Windows Security:** Since this is a custom-built app, Windows might show a "SmartScreen" warning. Click **"More Info"** and then **"Run Anyway"** to start the app.
